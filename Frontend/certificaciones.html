C/C++

C++ es un lenguaje de programación de propósito general con un amplio soporte de bibliotecas. Fue creado como una mejora del lenguaje C por Bjarne Stroustrup en 1979 en los Laboratorios Bell en Murray Hills, Nueva Jersey.

VENTAJAS

DESVENTAJAS
  • C++ ofrece portabilidad, lo que significa que es independiente de la plataforma.
  • Admite diferentes bibliotecas con funciones para escribir código rápidamente.
  • Proporciona un control completo sobre la gestión de la memoria.
  • C++ ofrece mayor velocidad que otros lenguajes de programación de propósito general.
  • C++ es más difícil de entender sintácticamente en comparación con lenguajes de programación más avanzados.
  • C++ no ofrece soporte para recolección de basura, por lo que los programadores deben gestionar la memoria manualmente. Si no se gestiona de forma eficiente, puede provocar el almacenamiento de datos redundantes, aumentando así el uso de memoria.
  • C++ no incluye soporte integrado para hilos. Esto supone una desventaja frente a otros lenguajes como Java, ya que complica considerablemente el proceso.

Puntuación Mínima Requerida para Acreditar: 35/40

Tiempo para realizar el examen: 2hrs

Costo de aplicación del examen: $4,000

Disponible a partir del: 5 de noviembre de 2025

C#

C# permite a los desarrolladores crear diversas aplicaciones de software que se ejecutan en el marco de trabajo .NET de Windows. Se utiliza para desarrollar sistemas alojados y embebidos. Es posible encontrar el uso de C# en aplicaciones web y de escritorio, así como en videojuegos. La similitud de C# con los lenguajes de la familia C es otra razón de su popularidad y facilidad de aprendizaje.

VENTAJAS

DESVENTAJAS
  • Tiene un recolector de basura incorporado y es seguro contra errores de tipeo.
  • Como lenguaje de programación orientado a objetos, C# produce aplicaciones más fáciles de probar y leer.
  • Permite dar respuesta a cualquier problema que surja y proporciona un enfoque más eficiente para escribir código.
  • Al ser un lenguaje de alto nivel, C# es más fácil de entender que un lenguaje de bajo nivel.
  • C# requiere compilación del código cada vez que se realizan cambios, lo que puede provocar errores y fallos.
  • El uso de bibliotecas .NET puede añadir otra capa de complejidad al aprendizaje de C#.
  • Dado que C# forma parte del framework .NET, necesita la plataforma Windows para su ejecución.

Puntuación Mínima Requerida para Acreditar: 20/25

Tiempo para realizar el examen: 1hr 30min

Costo de aplicación del examen: $3,500

Disponible a partir del: 30 de diciembre de 2025

JAVA

Java es una plataforma informática de lenguaje de programación creada por Sun Microsystems en 1995. Ha evolucionado desde sus humildes comienzos hasta impulsar una gran parte del mundo digital actual, ya que es una plataforma fiable en la que se crean muchos servicios y aplicaciones. Los nuevos e innovadores productos y servicios digitales diseñados para el futuro también siguen basándose en Java.

VENTAJAS

DESVENTAJAS
  • Orientado a elementos: te permite crear proyectos estándar y código reutilizable.
  • Java es un lenguaje distribuido, ya que proporciona una herramienta para compartir información y proyectos entre numerosos ordenadores, lo que mejora el rendimiento y la eficiencia del sistema.
  • Es multihilo, tiene la capacidad de realizar numerosas tareas simultáneamente.
  • Java fue diseñado para ser fácil de usar y, por lo tanto, es fácil de escribir, compilar, depurar y aprender que otros lenguajes de programación.
  • Es un lenguaje más lento en comparación con otros dialectos, ya que consume mucha memoria.
  • Java requiere una gran cantidad de memoria, especialmente en comparación con otros lenguajes como C y C++.
  • Los códigos Java son extensos, lo que significa que contienen numerosas palabras y frases largas y complejas difíciles de leer y comprender. Esto puede disminuir la claridad del código.

Puntuación Mínima Requerida para Acreditar: 6/8

Tiempo para realizar el examen: 1hr

Costo de aplicación del examen: $5

PYTHON

Python es un lenguaje de programación orientado a objetos de alto nivel y fácil de interpretar con una sintaxis fácil de leer. Ideal para la creación de prototipos y tareas ad hoc, Python tiene un amplio uso en computación científica, desarrollo web y automatización. Como un lenguaje de programación para principiantes de uso general, Python es compatible con muchos de los principales científicos informáticos y desarrolladores de aplicaciones de todo el mundo.

VENTAJAS

DESVENTAJAS
  • Python es fácil de aprender y genera una sensación de éxito en poco tiempo.
  • Python es muy versátil y flexible gracias a su diseño. Es multiplataforma por lo que puedes empezar un proyecto en un sistema operativo y continuar en otro.
  • Un gran número de pequeños desarrolladores, sino también para muchas empresas de mayor tamaño. Google, Spotify, Uber y Netflix dependen total o parcialmente de Python.
  • Python a veces tiene deficiencias de velocidad, especialmente en comparación con sus competidores.
  • Python tiene déficits notables en el desarrollo para dispositivos móviles, razón por la cual se escriben pocas aplicaciones móviles con este lenguaje de programación.
  • Aunque Python, en general, es muy seguro, su enfoque dinámico puede llevar a descubrir errores durante la ejecución.

Puntuación Mínima Requerida para Acreditar: 40/45

Tiempo para realizar el examen: 2hrs 30min

Costo de aplicación del examen: $4,500

Disponible a partir del: 28 de febrero de 2026
